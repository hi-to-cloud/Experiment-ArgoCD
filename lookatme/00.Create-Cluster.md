## Create EKS Cluster
→ Install awscli
```
aws --version
```
→ Install eksctl
```
eksctl version
```
→ Install Kubernetes-cli
```
kubectl version --client
```
→ Authenticate User with AWS
```
aws configure
```
```
aws sts get-caller-identity
```
```
{
    "UserId": "XGEGRHESHDRRHEG",

```
```
eksctl create cluster -f cluster.yml
```
